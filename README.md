## Implemented Features

- Calculator with `try`, `catch`, `finally`, `throw`, and a custom `InvalidCalculationError`.
- Buttons for required console methods, including log, error, table, dir, dirxml, group, time, trace, count, warn, assert, and clear.
- Global error handling with `window.addEventListener('error')`.
- "Trigger a Global Error" button that throws an intentional error and logs that it was caught.
- TrackJS monitoring script installed in `index.html`.
